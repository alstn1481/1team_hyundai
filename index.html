현대자동차 N은 모터스포츠 세계에서 축적한 경험을 통해 스릴과 감성적 즐거움을 장착한 모델로, 레이싱 트랙에서 느끼는 드라이빙 본연의 재미를 일상에서도 전달할 수 있도록 했습니다.

"기술력 남양", 주행력 뉘르부르크, N의 성능
"퍼포먼스 블루"

현대 N은 국내 최고의 자동차 기술의 산실인 현대 남양연구소의 기술력, 유서깊은 뉘르부르크링의 서킷에서 진행되는 실전테스트를 통해서만 만들어지는 차량들입니다.
기술력과 테스트, 차량 주행에 가장 중요한 두가지 요소를 무한히 반복함으로서, 현대 N은 드라이버 분들께 운전의 기쁨을 선사합니다.

괜찮아 우리들은 N이니까

현대 N은 N시리즈와 N line를 통해 고객분들께 일상의 여유로운 주행과 스릴넘치는 레이싱을 모두 제공해드리고 있습니다

갓성능의 N

N시리즈는 모터스포츠에서 다듬어진 고성능의 주행력을 일상속으로 끌어들인 시리즈입니다.
N브랜드의 상징인 고성능의 폭발적인 퍼포먼스가 그대로 적용된 N은 현대 기술력의 정점을 보여줍니다

주요기종: Avante N, Ioniq 5 N

갓성비의 NLine

N line은 N시리즈의 고성능 퍼포먼스와 기존 현대차량의 안정적 주행능력을 겸비한 상위 시리즈입니다.
N브랜드의 스포티한 디자인과 튜닝, 기존 현대차의 안정감을 모두 챙겼습니다.

주요기종: Sonata N, Tucson N
